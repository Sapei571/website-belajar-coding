Pelajari HTML dari Awal

Kuasai dasar-dasar HTML dengan contoh langsung, latihan, dan kuis.

apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. Ini menggambarkan struktur halaman web menggunakan elemen seperti judul, paragraf, dan tautan.

Pelajaran 1: Struktur Dasar HTML

Dokumen HTML memiliki struktur khusus yang mencakup deklarasi doctype, yaitu `html`, `head`, dan `body` elemen. Di bawah ini adalah struktur HTML paling dasar: <!DOCTYPE html> <html> <head> <title>nama website</title> </head> <body> <h1>untuk judul text website!</h1> </body> </html>

pelajaran 2: Heading dan Paragraphs

HTML memungkinkan Anda membuat berbagai tingkat judul (`h1` sampai `h6`) dan paragraf menggunakan `p` menandai. Berikut ini contohnya:

<h1>ini adalah Heading 1</h1>
<p> ini adalah paragraph.</p>
<h2> ini adalah Heading 2</h2>
<p>Ini adalah hal lain paragraph.</p>

Contoh di atas membuat dua judul dan dua paragraf pada halaman web.

pelajaran 3: Tables

Tabel HTML digunakan untuk menampilkan data dalam format terstruktur menggunakan baris dan kolom. Berikut ini contoh dasar tabel:

<table>
  <tr>
    <th>Nama</th>
    <th>umur</th>
  </tr>
  <tr>
    <td>samsudin hooh tenan</td>
    <td>40</td>
  </tr>
  <tr>
    <td>pabroro</td>
    <td>30</td>
  </tr>
</table>

Contoh ini memperlihatkan tabel dengan dua kolom dan tiga baris.

pelajaran 4: Forms

Formulir HTML memungkinkan Anda mengumpulkan masukan pengguna. Formulir dasar mencakup elemen seperti bidang teks, tombol radio, dan tombol kirim:

<form>
  <label for="nama">Nama:</label>
  <input type="text" id="nama" name="nama">
  <br>
  <label for="umur">umur:</label>
  <input type="number" id="umur" name="umur">
  <br>
  <input type="submit" value="Submit">
</form>

Pengguna dapat mengisi formulir dan mengirimkan datanya.

pelajaran 5: Multimedia

HTML memungkinkan Anda menyematkan elemen multimedia seperti gambar, video, dan audio. Berikut cara menambahkannya:

<img src="image.jpg" alt="nama untuk image">

<video controls>
  <source src="video.mp4" type="video/mp4">
  Browser Anda tidak mendukung tag video.
</video>

<audio controls>
  <source src="audio.mp3" type="audio/mp3">
  Browser Anda tidak mendukung elemen audio.
</audio>

Contoh ini menunjukkan cara menyematkan file gambar, video, dan audio dalam dokumen HTML.
