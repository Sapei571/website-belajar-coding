Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan elemen HTML.

Lesson 1: Struktur Dasar CSS

CSS digunakan untuk menata elemen HTML dengan cara menambahkan properti dan nilai. Contoh:

body {
    background-color: #f0f0f0;
}

Lesson 2: Selektor CSS

Selektor digunakan untuk memilih elemen HTML yang akan diberi gaya. Contoh:

h1 {
    color: blue;
}

Lesson 3: Box Model

Setiap elemen HTML memiliki model kotak yang terdiri dari margin, border, padding, dan konten.

div {
    margin: 10px;
    padding: 15px;
    border: 1px solid #000;
}

Lesson 4: Flexbox

Flexbox adalah cara modern untuk membuat layout responsif dengan mudah.

.container {
    display: flex;
    justify-content: space-between;
}
