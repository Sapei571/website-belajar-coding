Lesson 1: Variables and Data Types

JavaScript uses variables to store data. You can declare a variable using let or const. Example:

let name = "John";
let age = 25;
const isStudent = true;

In the example above, we create three variables: name, age, and isStudent.

Lesson 2: Functions

A JavaScript function is a block of code designed to perform a task. You can define a function and call it. Example:

function greet(name) {
  return "Hello " + name;
}
console.log(greet("Alice"));

In the example above, the function greet takes one argument and returns a greeting message.

Lesson 3: Conditionals

Conditional statements in JavaScript help you to make decisions based on conditions. Example:

let age = 20;
if (age >= 18) {
  console.log("You are an adult");
} else {
  console.log("You are a minor");
}

This code checks if the age is greater than or equal to 18 and prints the appropriate message.

Lesson 4: Loops

Loops allow you to execute a block of code multiple times. Example of a for loop:

for (let i = 0; i < 5; i++) {
  console.log(i);
}

This loop prints the numbers 0 to 4 in the console.

Lesson 5: Arrays

Arrays are used to store multiple values in a single variable. Example:

let colors = ["red", "green", "blue"];
console.log(colors[0]); // Outputs "red"

This example creates an array of colors and accesses the first item.
